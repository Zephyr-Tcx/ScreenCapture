# 🖥️ 定时屏幕截图工具

一个简单好用的 Windows 屏幕截图工具，可以定时自动截图！

## 主要功能

- **定时截图**：设置几分钟截一次图
- **手动截图**：随时点击按钮截图
- **多显示器支持**：可以选择截哪个屏幕
- **自定义保存位置**：图片存到哪里你说了算
- **系统托盘运行**：可以最小化到右下角

## 快速使用

### 直接下载
1. [Releases] 页面
2. 下载 `ScreenCapture.exe`
3. 双击运行

## 使用教程

1. **启动程序**：双击 ScreenCapture_v0.0.*.exe
2. **设置间隔**：输入数字（比如 5，表示5分钟）
3. **选择保存位置**：点击"浏览"选文件夹
4. **开始截图**：点击"开始截图"按钮
5. **手动截图**：随时点"立即截图"

## 文件保存
截图默认保存在：文档\ScreenCaptures\

配置保存在：文档\ScreenCaptures\settings.ini

图片命名格式：auto_YYYY-MM-DD_hh-mm-ss.png（自动）
manual_YYYY-MM-DD_hh-mm-ss.png（手动）

## 系统要求

Windows 7 或更高版本

无需额外安装运行时库（静态编译）

支持多显示器环境
