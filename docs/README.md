# 🖥️ 定时屏幕截图工具

一个简单好用的 Windows 屏幕截图工具，可以定时自动截图！

## ✨ 主要功能

- ⏰ **定时截图**：设置几分钟截一次图
- 🖱️ **手动截图**：随时点击按钮截图
- 🖥️ **多显示器支持**：可以选择截哪个屏幕
- 📁 **自定义保存位置**：图片存到哪里你说了算
- 📂 **系统托盘运行**：可以最小化到右下角

## 📦 快速使用

### 方法1：直接下载（最简单）
1. [Releases] 页面
2. 下载 `ScreenCapture.exe`
3. 双击运行！

### 方法2：自己编译
1. 下载源代码
2. 双击 `build.bat`
3. 等编译完成

## 🛠️ 使用教程

1. **启动程序**：双击 ScreenCapture.exe
2. **设置间隔**：输入数字（比如 5，表示5分钟）
3. **选择保存位置**：点击"浏览"选文件夹
4. **开始截图**：点击"开始截图"按钮
5. **手动截图**：随时点"立即截图"

## ❓ 常见问题

**Q：为什么截图失败？**
A：可能是权限问题，试试用管理员身份运行

**Q：保存的图片在哪里？**
A：默认在"文档\ScreenCaptures"文件夹
